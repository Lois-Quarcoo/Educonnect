# PDF Assets Folder

This folder contains PDF files for testing the AI features in Educonnect.

## How to Add Your Own PDF:

1. **Copy your PDF file** to this folder
2. **Make sure it has a .pdf extension**
3. **The app will be able to access it for testing**

## Current Files:

- `sample-educational-content.pdf` - Sample PDF for testing (placeholder)

## Testing AI Features:

Once you have PDF files in this folder, you can:
1. Upload them through the app
2. Generate AI summaries
3. Create study guides
4. Generate quizzes

## Recommended PDF Types for Testing:

- Educational content (textbooks, notes)
- Technical documentation
- Study materials
- Research papers

The AI features work best with text-based PDFs that have clear structure and content.
